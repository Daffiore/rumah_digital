# 🏠 Rumah Digital

Proyek aplikasi web berbasis framework **Laravel** yang mengintegrasikan sistem autentikasi modern dan dashboard manajemen aktivitas organisasi. Desain antarmuka dibangun menggunakan **Tailwind CSS v4** dengan pendekatan responsif (aman diakses via Mobile maupun Desktop).

---

## 🚀 Fitur Utama
- **Sistem Login Modern**: Halaman login estetik menggunakan gaya *glassmorphic card* dengan penanganan error yang andal.
- **Responsive Dashboard**: Layout dashboard menggunakan navigasi sidebar yang otomatis menyesuaikan skala ketika dibuka melalui smartphone.
- **Aktivitas Pengurus Carousel**: Menampilkan lini program kerja dan aktivitas organisasi (seperti PSDM-Akademik) dalam bentuk slider visual yang rapi.

## 🛠️ Teknologi & Library
- **Backend:** Laravel (PHP)
- **Frontend CSS:** Tailwind CSS v4 (via JSDelivr CDN)
- **Ikon & Grafis:** FontAwesome v6 & Google Fonts

---

## 📂 Struktur Branch Repositori
Proyek ini dibagi ke dalam beberapa branch untuk kerapian manajemen kode:
- `main` / `master` : Cabang produksi utama (backend & core logic).
- `front-end` : Cabang khusus pengembangan visual, slicing mockup, tata letak Blade, dan optimasi responsif mobile.
